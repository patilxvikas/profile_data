Projects

Pokedex

The Pokedex React App is a sleek and interactive application built with React, offering users comprehensive information about various Pokémon. It features a real-time search with autocomplete for quick access to Pokémon by name or number, along with detailed pages displaying each Pokémon's name, image, type, abilities, and stats. Designed to be responsive, the app ensures a seamless experience across desktop and mobile devices, complemented by a clean and intuitive user interface that supports both dark and light modes. Performance is optimized with lazy loading and caching techniques, making data retrieval efficient and fast. Powered by React, React Router for navigation, Axios for API calls, and Styled Components for modular CSS, this app is ideal for Pokémon enthusiasts seeking a user-friendly tool to explore and learn about their favorite creatures.

8 Puzzle Solver

The 8-bit Puzzle Project is an engaging web application crafted using HTML, CSS, and JavaScript, featuring an implementation of the A* search algorithm for pathfinding. This nostalgic-inspired game challenges users to solve puzzles by maneuvering an 8-bit character through a grid-based maze. The game interface, styled with a retro aesthetic using CSS, immerses players in a pixelated world reminiscent of classic video games. Leveraging JavaScript, the A* search algorithm intelligently navigates through obstacles, calculating the shortest path from the starting point to the goal in real-time, providing an interactive and dynamic gameplay experience. Whether you're a puzzle enthusiast or a fan of retro gaming, the 8-bit Puzzle Project promises hours of strategic fun while showcasing the power of A* search in solving pathfinding challenges within a nostalgic digital environment.
